Ghoomo Saste Me - Full-Stack Travel Portal
Ghoomo Saste Me is a professional, pocket-friendly travel booking platform designed for students and middle-class travelers. This project is a full-stack web application built using React.js, Node.js, Express, and MySQL. It features dynamic trip management, interactive UI components, and a specialized itinerary routing system.

 Key Features Implemented
1. Dynamic Trip Management
Database Driven: All trip data (Destination, Price, Category, Image) is fetched dynamically from a MySQL database.

Category Filtering: Trips are organized into sections like Himachal, Uttarakhand, and Weekend Getaways for easy navigation.

2. Interactive & Responsive UI
High-Impact Hero Section: Features a background video with a real-time search bar ("Where do you want to go?").

Offer Popup: A lead-generation popup highlighting the "Explore India @ ₹5000" branding to capture user interest.

Category Navigation: Icon-based navigation for Backpacking, Honeymoon, Corporate, and Adventure trips.

3. Smart Routing & Itineraries
Single Trip Pages: Built with react-router-dom, each trip card links to a unique itinerary page.

Detailed Itineraries: Displays day-wise plans, inclusions, and specialized pricing for budget travelers.

4. Trust & Conversion
Comprehensive FAQ: A dedicated section answering 13 critical budget travel questions to build user trust.

Direct Inquiry: Integrated "Book Now" buttons that facilitate direct communication via WhatsApp/Call.

Professional Footer: Contains office address (Gurugram), social links, and mandatory travel policies.
 Tech Stack
Frontend: React.js, CSS3 (Flexbox/Grid), Axios, React Router.

Backend: Node.js, Express.js.

Database: MySQL (Relational data storage).

Tools: Git Bash, VS Code, MySQL Workbench.
